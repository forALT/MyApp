### Git 명령어

    1. git 초기화 
        $ git init
    2. status 확인
        $ git status
    3. config 목록 확인
        $ git config --list
    4. github의 username과 email 설정
        $ git config --global user.name  각자의username
        $ git config user.name

        $ git config --global user.email  각자의email
        $ git config user.email
